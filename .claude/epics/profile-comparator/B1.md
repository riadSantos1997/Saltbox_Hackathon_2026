# B1 · Diff Engine (Pure Function)

## 🎯 Objective
Given two `ScrapeResult[]` arrays (one per org), compute a flat `DiffRow[]` of differences — absences and value mismatches only. Pure, unit-testable, no IO.

## 📂 Files
- `lib/diff/engine.ts`

## ✅ Acceptance Criteria
- [ ] Exports `diff(a: ScrapeResult[], b: ScrapeResult[]): DiffRow[]`.
- [ ] `DiffRow` includes: `key`, `category`, `valueA`, `valueB`, `type` ∈ `missing_in_a` | `missing_in_b` | `value_mismatch`.
- [ ] Rows where `valueA === valueB` are **excluded** — per PRD, XLS contains only differences.
- [ ] Returns empty array when orgs are identical → D4 uses this to message "no differences" without generating a file (→ AC-05).
- [ ] Comparison is deep-equal for boolean and string values.

## 🔗 Dependencies
- **Depends on:** A3 (real `ScrapeResult` shape); can also be started earlier against stubbed data if `types.ts` (A1) is committed.
- **Blocks:** B2.

## 📝 Implementation Sketch
```ts
export function diff(a: ScrapeResult[], b: ScrapeResult[]): DiffRow[] {
  const rows: DiffRow[] = [];
  const aByKey = new Map<string, PermissionRow>();
  const bByKey = new Map<string, PermissionRow>();
  a.forEach(r => r.rows.forEach(row => aByKey.set(`${r.category}:${row.key}`, row)));
  b.forEach(r => r.rows.forEach(row => bByKey.set(`${r.category}:${row.key}`, row)));
  const keys = new Set([...aByKey.keys(), ...bByKey.keys()]);
  for (const k of keys) {
    const av = aByKey.get(k), bv = bByKey.get(k);
    if (!av) rows.push({ key: k, ...bv, type: 'missing_in_a', valueA: null, valueB: bv!.values });
    else if (!bv) rows.push({ key: k, ...av, type: 'missing_in_b', valueA: av.values, valueB: null });
    else if (JSON.stringify(av.values) !== JSON.stringify(bv.values)) rows.push({ ... type: 'value_mismatch' });
  }
  return rows;
}
```

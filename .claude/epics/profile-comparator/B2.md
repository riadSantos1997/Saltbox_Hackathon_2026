# B2 · XLS Generator + Export Route

## 🎯 Objective
Take `DiffRow[]` from B1, emit a single-sheet XLSX with the 5 columns per PRD, yellow for mismatches, red for missing. Serve via `/api/export` with the right content-type and filename. Implements **F-04 / AC-05**.

## 📂 Files
- `lib/xlsx/generator.ts` — `generateXlsx(rows, meta): Buffer`
- `app/api/export/route.ts` — `POST` that accepts diff payload and returns the file

## ✅ Acceptance Criteria (→ AC-05)
- [ ] Single sheet named "Differences".
- [ ] Columns exactly: `Permission / Object`, `Category`, `Org A (Reference)`, `Org B`, `Difference Type`.
- [ ] Difference Type cell text: `"Missing in Org A"` | `"Missing in Org B"` | `"Value mismatch"`.
- [ ] Row fill: value_mismatch → yellow (`FFFFFF00`); missing → red (`FFFF0000`).
- [ ] Filename from response header: `profile-comparison_{ProfileName}_{OrgA}_vs_{OrgB}_{YYYY-MM-DD}.xlsx` — passed via `Content-Disposition`.
- [ ] Empty diff → route returns HTTP 204 (D4 reads this and shows "No differences found" message instead of a download link).
- [ ] Uses `xlsx` npm package (SheetJS) — already in stack.

## 🔗 Dependencies
- **Depends on:** B1.
- **Blocks:** D4.

## 📝 Notes
- Colors are enhancement, not primary signal — Difference Type text column is authoritative (addresses Medium risk: XLS color rendering).
- Consider using `xlsx-js-style` (a `xlsx` fork) if cell-level fills are a struggle in plain `xlsx`.

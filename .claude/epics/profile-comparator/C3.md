# C3 · Object Picker (Object Settings only)

## 🎯 Objective
Inline searchable checkbox list of Salesforce objects. Renders only when the user selects `Object Settings` in C2. Implements **F-03 / AC-04**.

## 📂 Files
- `app/components/selectors/ObjectPicker.tsx`

## ✅ Acceptance Criteria (→ AC-04)
- [ ] Fetches from `/api/salesforce/objects` on mount; renders within 3s of selection.
- [ ] Search input filters client-side on each keystroke — no server roundtrip.
- [ ] `Select All` shortcut checks every currently-visible (filtered) item.
- [ ] `Run Comparison` CTA disabled until ≥1 checkbox checked.
- [ ] Virtualizes with `react-window` when list > 50 items (addresses DOM overload).
- [ ] Emits `onConfirm: (selectedApiNames: string[]) => void`.

- **Blocks:** D4.
